## Working with Shell 02 ##

### WHAT: ###
Text Editors
- vi editor: most popular
    - command mode
    - insert mode
    - last line


---

### WHY: ###
- Create / Modified files
---

### HOW: ####
- See below
---

```bash
# vi start at command mode
# copy or delete file
[~]$ vi /home/bob/sample.txt

# Change to insert mode
# Press "i" 
# add, remove, and edit file
# esc key to go back to command mode

# Change to the last line mode
# press ":"
# Save file, discard changes and
# exit vi editor

# Move Around: h,k,j,l
# Copy a line: y y
# Paste a line: pp 
# Delete a letter: x
# Delete a line: dd
# Delete 3 lines: d3d
# Save a file: ZZ
# Undo: u
# Redo: Ctrl r
# Find pattern: /line ?line
# Find Next: n
# Find Prev: N

#Insert Mode
# type: i,o,a --insert--
# esc
# Save :w
# Quit :q
# Save & Quit :wq
# Quit :q!







```
